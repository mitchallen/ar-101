ar-101
==

## Usage

On iOS, browse to this site:

* https://mitch-ar.surge.sh

It's still a bit wonky.

* Tap the screen to get the object menu
* Pinch and zoom to try to place the object.

## Local host

```sh
python3 -m http.server
```

In a new window:

```sh
open http://localhost:8000
```
